# Complete Guide to Reading Query Strings in NestJS

A query string is the part of a URL after the ? mark, used to send data to the server.

```
# URL Structure
https://example.com/users?page=2&limit=10&sort=asc
                       └─────────┬─────────┘
                           Query String

```

Accessing Query Parameters

```typescript

// users.controller.ts
import { Controller, Get, Query } from '@nestjs/common';

@Controller('users')
export class UsersController {
  @Get()
  getAllUsers(@Query() query: any) {
    // query contains all query parameters
    console.log('Full query object:', query);
    return {
      message: 'Users fetched successfully',
      query: query
    };
  }
}

```


```

# Request
GET http://localhost:3000/users?page=2&limit=10&sort=asc

# Response
{
  "message": "Users fetched successfully",
  "query": {
    "page": "2",
    "limit": "10",
    "sort": "asc"
  }
}

```

Basic Access - Method 2: Specific Parameters


```typescript

// users.controller.ts
import { Controller, Get, Query } from '@nestjs/common';

@Controller('users')
export class UsersController {
  @Get()
  getAllUsers(
    @Query('page') page: string,
    @Query('limit') limit: string,
    @Query('sort') sort: string
  ) {
    return {
      page: page || '1',      // Default value if not provided
      limit: limit || '10',
      sort: sort || 'asc'
    };
  }
}

```


```

# With all parameters
GET /users?page=2&limit=20&sort=desc
# Response: { "page": "2", "limit": "20", "sort": "desc" }

# Without parameters
GET /users
# Response: { "page": "1", "limit": "10", "sort": "asc" }

# With partial parameters
GET /users?page=3
# Response: { "page": "3", "limit": "10", "sort": "asc" }

```

# Types of Dependencies

A dependency is something that a class or function needs to work.

Types of Dependencies:
⭐ 1. Direct Dependency (Tight Coupling)
⭐ 2. Dependency Injection (Loose Coupling)
⭐ 3. External Dependency
⭐ 4. Optional Dependency
⭐ 5. Runtime Dependency

## Direct Dependency (Tight Coupling)

When a class creates its dependency itself, it becomes tightly coupled.

```

class UserService {
  logger = new LoggerService();

  createUser() {
    this.logger.log("User created");
  }
}

```

⭐ 1 Constructor Injection (Most Used)

```
constructor(private userService: UserService) {}

```

⭐ 2 Property Injection

```
@Inject(UserService)
private userService: UserService;

```

## Dependency Injection (Loose Coupling)
Instead of creating dependency inside class → Dependency is given from outside

```
class UserService {
  constructor(private logger: LoggerService) {}

  createUser() {
    this.logger.log("User created");
  }
}

```

## External Dependency

These are dependencies that come from outside libraries.

Examples:
   - axios
   - bcrypt
   - typeorm
   - pg
   - class-validator


## Optional Dependency

Sometimes dependency may or may not exist.

```
constructor(@Optional() private cacheService?: CacheService) {}

```

## Runtime Dependency
Dependency needed only during execution.

Example:
- database connection
- redis connection
- message broker
